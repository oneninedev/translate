## 클립보드 번역

#### 단축키로 번역기 페이지를 실행합니다.
#### 이때 내용은 클립보드의 내용이 번역됩니다.
#### 

## 단축키

#### 한글 번역
#### Mac : cmd + k
#### Window : ctrl + k
#### 영어 번역
#### Mac : cmd + e
#### Window : ctrl + e

### 기술
- electron.js
### 배포 명령어
yarn dist
